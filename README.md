# Estatística Descritiva — UEPB

Ferramenta de Estatística Descritiva desenvolvida para a UEPB, focada na análise de distribuições de frequência discreta. O sistema organiza dados, realiza cálculos estatísticos automaticamente e gera tabelas interativas com abordagem didática para apoio acadêmico.

---

## Funcionalidades

- Organização automática dos dados em ordem crescente (Rol)
- Geração de tabelas de distribuição de frequência
- Cálculo automático de:
  - Frequência absoluta (`fi`)
  - Frequência relativa simples (`frs`)
  - Frequência percentual simples (`fps%`)
  - Frequência acumulada crescente (`fac`)
  - Frequência acumulada decrescente (`fad`)
  - Frequência percentual acumulada crescente (`fpa%`)
  - Frequência percentual acumulada decrescente (`fpda%`)
- Resumo estatístico:
  - Total de observações (`n`)
  - Valor mínimo
  - Valor máximo
  - Amplitude total (`AT`)
- Tabela dinâmica com colunas opcionais
- Suporte a números inteiros e decimais
- Sanitização automática de entrada
- Glossário integrado com fórmulas e notações estatísticas
- Interface didática com foco acadêmico
- Feedback visual e animações de interação

## Objetivo do projeto

O projeto foi desenvolvido para auxiliar estudantes da UEPB no estudo de Estatística Descritiva, especialmente na compreensão de distribuições de frequência e organização de dados estatísticos.

A ferramenta busca aproximar teoria e prática através de visualização interativa e automatização de cálculos estatísticos.

---

## Estrutura estatística implementada

| Símbolo | Descrição |
|---|---|
| `n` | Total de observações |
| `xi` | Valor observado |
| `fi` | Frequência absoluta |
| `frs` | Frequência relativa simples |
| `fps%` | Frequência percentual simples |
| `fac` | Frequência acumulada crescente |
| `fad` | Frequência acumulada decrescente |
| `fpa%` | Frequência percentual acumulada crescente |
| `fpda%` | Frequência percentual acumulada decrescente |
| `AT` | Amplitude total |

> Obs.: Em distribuições contínuas, `xi` representa o ponto médio de cada classe.

---

## Como utilizar

1. Insira os valores separados por:
   - espaço
   - quebra de linha
2. Clique em `Calcular ƒ`
3. Visualize:
   - Rol ordenado
   - Resumo estatístico
   - Distribuição de frequências
4. Ative ou desative colunas conforme necessário
