Notação e fórmulas

n

Total de observações

Quantidade total de valores no conjunto de dados.

xi

Valor observado

Cada valor distinto presente no conjunto de dados. Obs: na distribuição contínua, xi representa o ponto médio de cada classe.

fi

Frequência absoluta

Quantas vezes cada valor xi aparece no conjunto.

frs

Frequência relativa simples

Proporção de cada valor em relação ao total.

frs = fi / n

fps%

Frequência percentual simples

Frequência relativa expressa em porcentagem.

fps = (fi / n) × 100

fac

Frequência acumulada crescente

Soma das frequências absolutas até o valor xi.

fac = fi₁ + fi₂ + … + fiₖ

fad

Frequência acumulada decrescente

Soma das frequências absolutas a partir do valor xi até o maior.

fad = fiₖ + … + fiₙ

fpa%

Freq. percentual acumulada crescente

Frequência acumulada crescente expressa em porcentagem.

fpa = (fac / n) × 100

fpda%

Freq. percentual acumulada decrescente

Frequência acumulada decrescente expressa em porcentagem.

fpda = (fad / n) × 100

AT

Amplitude total

Diferença entre o maior e o menor valor do conjunto.

AT = Xmáx − Xmín
